<h1>JActive Directory Lab</h1>



<h2>Description</h2>
In this video, I went through how to create an Active Directory home lab environment, set up a Domain Controller, and connect to it with a client computer using any account you create in the Domain. Thanks for watching :)
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
